# Marketing Campaign Performance Dashboard

## Project Overview
This project analyzes multi-channel marketing campaign data using Excel and Google Sheets.
An interactive dashboard was built to track performance across channels, regions, and time.

## Key KPIs
- Total Spend
- Total Revenue
- Total Profit
- Average ROAS
- Conversion Rate

## Dashboard Features
- Monthly Spend vs Revenue Trend
- ROAS by Channel
- Conversions and Profit by Region
- Marketing Funnel (Impressions → Clicks → Conversions)
- Budget Distribution by Channel
- Interactive Slicers (Channel, Region, Month)

## Tools Used
- Google Sheets / Excel
- Pivot Tables
- KPI Calculations (CTR, CPC, CPA, ROAS, Profit, Conversion Rate)
- Data Visualization & Dashboard Design

## Business Insights
- Instagram delivers the highest ROAS.
- Google Ads generates the highest revenue.
- West region shows the highest conversions and profit.
- Marketing funnel shows scope to improve CTR and conversion rate.
